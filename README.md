# twitter-clone-for-softcom-test
A Twitter clone for a tech interview with Softcom

## Features:
- User signup
- User sign in (using JWT)
- Post tweet
- Reply to tweet
- Follow other users
- View own timeline
- Search (Tweets and Users)

### Bonus:
- Unit testing (CI/CD)
- Real-time timeline update
